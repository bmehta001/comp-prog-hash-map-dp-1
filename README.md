# Hash Maps, Dynamic Programming & Data Structures

A competitive programming assignment focusing on hash map internals,
dynamic programming techniques, and linked list implementation.

## Questions

| # | Name | Topic | Difficulty |
|---|------|-------|------------|
| 1 | Custom HashMap | Hash maps / open addressing / resizing | ⭐⭐⭐ |
| 2 | Custom Linked List | Linked lists / pointers | ⭐⭐⭐ |
| 3 | Fibonacci | Recursion → memoization → DP → iteration + analysis | ⭐⭐ |
| 4 | Factorial | Recursion → memoization → DP → iteration + analysis | ⭐⭐ |
| 5 | Climbing Stairs | Recursion → memoization → iteration | ⭐ |
| 6 | Coin Combination Amounts | Recursion → memoization → DP | ⭐⭐ |

## Getting Started

See [INSTRUCTIONS.md](INSTRUCTIONS.md) for setup, compilation, and
submission details.

## Languages

Each question has implementations in **Java**, **Python**, and **C++**.
You only need to complete **one language** per question.

| Language | Directory | Build Command |
|----------|-----------|---------------|
| Java | `java/` | `javac *.java` |
| Python | `python/` | `python questionN_*.py` |
| C++ | `cpp/` | See CMakeLists.txt or compile individually |
